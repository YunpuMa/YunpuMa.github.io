Change GitHub PAT via terminal
https://USERNAME:YOUR_PAT_TOKEN@github.com/YOUR_USERNAME/YOUR_REPO.git
Replace the following placeholders with your information:
USERNAME: Your GitHub username.
YOUR_PAT_TOKEN: The Personal Access Token you generated.
YOUR_USERNAME: Your GitHub username again.
YOUR_REPO: The name of your repository.
